## Image Guidelines

1. **Format**: PNG or WebP preferred
2. **Size**: Keep under 500KB for fast loading
3. **Resolution**: 2x for retina displays
4. **Alt text**: Always add descriptive alt text in markdown

## Quick Capture Tips

### Windows
- `Win + Shift + S` - Snipping Tool
- ShareX for advanced captures

### Terminal
- Use Windows Terminal for better screenshots
- Consider dark theme for consistency

## Adding Images to Markdown

```markdown
![Description](images/filename.png)
```

With link:
```markdown
[![Description](images/filename.png)](https://link-url.com)
```
